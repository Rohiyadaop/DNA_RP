## 🚀 QUICKEST PATH (Under 5 Minutes)

```bash
# Terminal 1 - Backend
cd backend
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env
# ⚠️ Edit .env - add your NVIDIA_API_KEY
uvicorn app:app --reload

# Terminal 2 - Frontend
cd frontend
npm install
cp .env.example .env.local
npm run dev

# Browser
# Visit: http://localhost:3000
# Click "Load example" and "Generate DNA Sequence"
```

**That's it!** You're running BioGPT.
